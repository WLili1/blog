# 个人博客系统

### 博客系统功能
1. 后台管理
    - 管理员登录
    - 博客管理
    - 分类管理
    - 标签管理

2. 前端展示
    - 首页展示
    - 分类页展示
    - 标签页展示
    - 归档页展示
    - 关于我

### 技术组合
    - 后台：springboot2.0 + JPA + thymeleaf
    - 数据库：MySQL5.7
    - 前端ui： Semantic UI

### 工具以及环境
    - IDEA
    - Maven3
    - JDK8
